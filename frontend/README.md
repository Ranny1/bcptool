# BCPTool — Frontend

React + Vite frontend for the BCPTool application.

## Getting Started

```bash
npm install
npm run dev
```

The frontend connects to the FastAPI backend at `http://localhost:8000`.

## Tech Stack

- React 18 + Vite
- react-flow (graph editor for dependencies)
- Recharts (dashboards)
- Material-UI (components)
- React Query (API state)
- Zustand (local UI state)
